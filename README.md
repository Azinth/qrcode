# QRCode Swift

Um gerador de QR Code rápido e intuitivo construído com Next.js, Genkit e ShadCN UI.

## Como adicionar ao seu GitHub

Para subir este projeto para o seu próprio repositório no GitHub, siga estes passos:

1. **Crie um novo repositório vazio** no seu GitHub.
2. **Abra o terminal** na pasta raiz deste projeto.
3. **Execute os seguintes comandos**:

```bash
# Inicialize o repositório git local
git init

# Adicione todos os arquivos
git add .

# Faça o primeiro commit
git commit -m "feat: inicializando projeto QRCode Swift"

# Conecte ao seu repositório remoto
# (Substitua a URL abaixo pela URL do repositório que você criou)
git commit -m "feat: inicializando projeto QRCode Swift" https://github.com/SEU_USUARIO/SEU_REPOSITORIO.git

# Renomeie a branch para main
git branch -M main

# Envie o código para o GitHub
git push -u origin main
```

## Tecnologias Utilizadas

- **Framework:** Next.js 15 (App Router)
- **UI:** Tailwind CSS + ShadCN UI
- **Ícones:** Lucide React
- **Biblioteca de QR Code:** `qrcode`
- **Inteligência Artificial:** Genkit (Google AI)
