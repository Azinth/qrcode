# **App Name**: QRCode Swift

## Core Features:

- Text Input: Allows users to input text via a text field.
- QR Code Generation: Generates a QR code from the text entered.
- Display QR Code: Show the QR code in a view after it's generated.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5), offering a sense of reliability.
- Background color: Light gray (#F5F5F5), providing a clean, uncluttered backdrop.
- Accent color: Soft indigo (#5C6BC0), complementing the primary color with a touch of sophistication.
- Body and headline font: 'Inter' for a modern and neutral look.
- Use simple and clear icons for actions.
- A clean and straightforward layout with a central focus on the QR code.
- A subtle animation to confirm the generation of the QR code.
